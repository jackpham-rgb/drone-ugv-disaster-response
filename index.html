Trung Hieu Pham — Cypress College  ·  Dr. Adam Thorpe — Center for Autonomy, UT Austin  ·  Dr. Ufuk Topcu — Dept. of Aerospace Engineering, UT Austin

This project develops a two-tier cooperative multi-agent system for autonomous disaster response. Unmanned aerial vehicles (UAVs) using MAPPO perform distributed area exploration, writing discoveries to a shared belief map that guides ground robots (UGVs) using QMIX with an embedded fairness reward toward survivor rescue. A cellular automaton models dynamic fire spread. The system is evaluated across procedurally generated mountain and urban terrains.

Algorithm architecture
ComponentAlgorithmRoleKey mechanism
UAV policy MAPPO 2024 Exploration & discovery Centralized critic, decentralized actors. Coordination penalty prevents clustering without explicit communication.
UAV alt. policy Boids + RL 2024 Emergent flocking Separation, alignment, cohesion behavioral priors bias action selection. No central planner required.
UGV policy QMIX 2018→ Navigation & rescue Value decomposition: global Q = monotone mix of per-agent Q_i. Fairness term embedded in local Q-value.
Fairness reward FE reward Original paper Load balancing r_t^i = (ε + |e_t^i / ē_t − 1|) / ē_t — carried forward from REU paper into QMIX Q-value.
Pathfinding A* (terrain-weighted) Collision avoidance Movement cost varies by terrain type. UGVs route around fire. Drones fly at fixed altitude.
Environment Cellular automaton New Fire spread model P(ignite) = rate × fuel × wind_factor × neighbor_state. Wind direction randomized per seed.
Fairness reward — connection to original paper
r_t^i = (ε + |e_t^i / ē_t − 1|) / ē_t
r_t^i — reward for agent i at timestep t  ·  ε = 1e-7 — numerical stability constant
e_t^i — performance of agent i (inverse distance to nearest assigned target)
ē_t — mean performance across all n agents at time t

In the original paper (2024), this reward drove FE-MADDPG policy gradients. In this system, it is embedded directly into the QMIX per-agent Q-value function, combining value decomposition with the same fairness objective — a novel integration.
